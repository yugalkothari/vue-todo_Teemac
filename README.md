# vue-todo_Teemac
a simple todo app built on vue js and bootstrap.

-add task
-delete task
-edit task
-task categories done/undone.
-sticky alerts 
-sticky task statistics
-sticky task progressbar


Demo can be seen here http://yugalkothari.in/projects/vue-todo-bootstrap
